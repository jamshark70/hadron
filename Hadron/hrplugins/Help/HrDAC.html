HrDAC Hadron plugin that writes audio to hardware output busses


Inherits from: Object : HadronPlugin


This Hadron plugin is shipped with the Hadron distribution. Hadron plugins are not meant for instantiating by hand in code, they are used internally by the Hadron system.


This simple plugin writes audio to your stereo hardware outputs 0 and 1. You can also employ a Limiter before the sound reaches to the output.


Modulatable parameters: level


Inputs: Left and right channels for audio.


The level slider uses the HrSlider widget which is midi aware and automatable. For details, see the help file for HrSlider.
